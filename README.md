# Sistema de Leilões

## Sobre o projeto

Este projeto consiste em um sistema de leilões desenvolvido como atividade prática para aprendizado de versionamento de código utilizando Git e GitHub.

O sistema permite o gerenciamento de itens cadastrados em uma casa de leilões.

## Tecnologias utilizadas

* Java
* MySQL
* Git
* GitHub
